### 🚀 Quick Start Guide

#### Installation & Setup
```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Validate setup
python setup.py

# 3. Run quick demo
python demo.py

# 4. Launch interactive analysis
jupyter notebook main.ipynb
```

#### Usage Options
1. **Interactive Notebook** (Recommended): `main.ipynb`
2. **Direct Module Import**: Import functions from `modules/`
3. **Quick Demo**: Run `demo.py` for overview
